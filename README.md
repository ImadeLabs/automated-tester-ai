# 🚀 AI QA Automation Agent (Powered by Oxlo.ai)

## 📌 Overview
AI QA Automation Agent is a tool that automatically analyzes any website and generates:

- ✅ Functional Test Cases  
- ⚠️ Edge Case Scenarios  
- 🧪 Cypress Automation Scripts  
- 🐞 Bug & UX Issue Detection  

This helps QA engineers and developers save hours of manual testing effort.

---

## 🧠 Powered by Oxlo.ai

This project uses Oxlo.ai API to:

- Analyze webpage content
- Generate structured QA outputs
- Produce real Cypress test scripts

Model used:
- `oxlo-large`

---

## ⚙️ Tech Stack

- Frontend: React.js
- Backend: Node.js (Express)
- AI: Oxlo API
- Web Scraping: Cheerio
